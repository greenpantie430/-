**宝丝专用豆瓣自动顶帖程序（测试中）**  
使用脚本自动顶帖存在被豆瓣官方封禁账号的风险。本文档提供的方法仅用于技术交流建议优先通过手动参与互动，脚本仅作为辅助。

* 环境准备  
  * 确保你的电脑已安装 Python 3.x 环境，并在终端（Windows CMD/PowerShell 或 macOS Terminal）输入以下信息安装需要的工具：  
    * pip install requests lxml

* 获取账号凭证 (Cookie & CK)  
  * 为了让脚本模拟你的浏览器行为，需要获取以下两项信息：  
    * 获取 Cookie  
      1. 步骤1: 打开浏览器登录 豆瓣，进入目标帖子页面。  
         * windows:按F12打开开发者工具,选择网络标签页。  
         * Mac:触控板右键打开开发者工具,选择网络标签页  
      2. 步骤2: 刷新页面，点击第一个请求（通常为当前页面地址）。  
         * 在 Headers 下方找到 Request Headers，复制 Cookie: 后面的完整字符串。  
    *  获取 4 位 CK 码  
      1. 步骤1: 在开发者工具中，切换到应用标签页。  
      2. 步骤2: 点击cookie,在名称中找到4位ck  
* 脚本配置说明  
  * 直接修改脚本中的配置部分，将账号信息填入对应的位置：

if \_\_name\_\_ \== '\_\_main\_\_':  
    \# 配置目标帖子 ID (从浏览器地址栏直接复制 ID 数字)  
    ID \= "38500062"   
    \# 粘贴获取到的完整 Cookie 字符串  
    COOKIE \= "bid=...; ...; ck=xYSy;"      
    \# 粘贴获取到的 4 位 CK  
    MY\_CK \= "xYSy"   
    \# 剩下的逻辑无需修改，直接运行即可
